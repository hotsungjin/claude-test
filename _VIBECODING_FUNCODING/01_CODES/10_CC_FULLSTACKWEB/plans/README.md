# 📁 Plans Directory

이 폴더는 AICOM 프로젝트의 모든 계획, 문서, 가이드를 포함합니다.

## 📂 폴더 구조

### `/wireframes`
- **main.xml** - 메인 페이지 구조
- **header.xml** - 헤더 레이아웃
- **login.xml** - 로그인 페이지
- **signup.xml** - 회원가입 페이지  
- **board.xml** - 게시판 상세 페이지

### `/` (루트)
- **ARCHITECTURE.md** - 시스템 아키텍처 및 기술 스택
- **TESTDATA.md** - 테스트 데이터 정의
- **TESTPLAN.md** - 테스트 계획 및 시나리오
- **DEPLOYMENT.md** - AWS EC2 배포 가이드
- **HTTPS_IN_AWS.md** - CloudFront HTTPS 설정 가이드
- **PROGRESS.md** - 개발 진행 상황 및 변경 이력
- **NOTE.md** - 개발 중 발생한 이슈 및 해결 방법

## 🎯 각 파일의 용도

### 개발 전 참고
- `ARCHITECTURE.md` - 전체 구조 파악
- `wireframes/*.xml` - UI 레이아웃 확인

### 개발 중 참고
- `TESTDATA.md` - 초기 데이터 설정
- `NOTE.md` - 문제 해결 방법

### 배포 시 참고
- `DEPLOYMENT.md` - EC2 배포
- `HTTPS_IN_AWS.md` - HTTPS 설정

### 진행 상황 추적
- `PROGRESS.md` - 변경 사항 기록
